Simple git: Glossary

Simple explanations of Git things.

Branch

A collection of commits.

Commit

A snapshot of your repository. Contains extra information like when the snapshot was taken and a (hopefully) helpful message.

Remote

A server that talks Git, and maintains a version of your repository.

Repository

A directory on your computer that Git keeps track of.
